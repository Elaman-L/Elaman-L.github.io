Калькулятор

Этот калькулятор предоставляет следующие возможности:
  • Добавлять ковры с указанием ширины и высоты..
  • Вычислять площадь каждого ковра.
  • Отображать общую площадь всех ковров.
  • Вычислять стоимость всех ковров (80 сом за м²).

Добавление ковра:

Следуйте инструкциям выше, чтобы добавить ковер в таблицу.

Нажмите «Удалить», чтобы удалить последний добавленный ковер из таблицы.

Очистка таблицы:

Нажмите "Очистить", чтобы удалить все ковры из таблицы и сбросить расчеты.
